# backend-api

##Run Commands
##Step 1:clone the repo.
##Step 2:Run command npm i.
##Step 3:Run command npm start.

##Basic Informations:

BASE URL: localhost:5000/api/users/

/create
Create User,
method:POST

/update/:id
Update User,
method:PUT


/delete/:id
Delete User,
method:DELETE


/
Get all Users,
method:"GET"

/coordinates/?lng=<value>&lat=<value>
 Get users between that range,
  method:"GET"
